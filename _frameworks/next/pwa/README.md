# Next.js PWA Documentation Index

This directory breaks our Progressive Web App guidance into focused references that align with the rest of `_docs`.

- `overview.md` — high-level summary of Deal Scale’s PWA goals and guiding principles.
- `configuration.md` — manifest, service-worker, security, and secrets checklist.
- `runtime-patterns.md` — push, offline, install UX, update detection, and network-aware rendering.
- `testing.md` — Vitest/Playwright commands and expectations.
- `capability-exploration.md` — external inspiration, including What PWA Can Do Today demos.
- `mdn-coverage.md` — MDN topic map for guides, how-tos, tutorials, and APIs.
- `roadmap.md` — forward-looking enhancements tied to platform capabilities.

Update these notes whenever PWA-related features change or new platform support becomes available.
